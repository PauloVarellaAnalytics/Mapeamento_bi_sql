Mapeamento de Relatórios Power BI + SQL

Este projeto apresenta uma documentação técnica e visual sobre o processo de mapeamento de relatórios corporativos desenvolvidos em Power BI, conectados a fontes de dados via SQL Server e ETL.

Etapas realizadas:

  • Extração de tabelas com Diagnóstico de Desempenho e Power BI Helper
  • Identificação de servidores e bancos de dados de origem
  • Mapeamento de procedures e jobs relacionados
  • Associação com projetos/pacotes ETL
  • Verificação de comandos INSERT INTO em procedures para validação da origem dos dados
